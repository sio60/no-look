# 🎙️ OBS 오디오 세팅 가이드 (No-Look)

Zoom 회의 소리를 우리 AI(`stt_core.py`)에 전달하기 위한 필수 설정입니다.

## 1. 사전 준비
- **VB-Cable** 드라이버가 설치되어 있어야 합니다.
- **OBS Studio**가 설치되어 있어야 합니다.

## 2. OBS 오디오 설정
1. OBS 실행 -> 우측 하단 **[설정]** 클릭.
2. 좌측 메뉴 **[오디오]** 탭 클릭.
3. **[고급(Advanced)]** 섹션 찾기.
4. **모니터링 장치 (Monitoring Device)**를 `CABLE Input (VB-Audio Virtual Cable)`으로 변경.
5. [적용] 및 [확인].

## 3. Zoom 소리 캡처 설정
1. OBS 메인 화면 하단 **[소스 목록]** -> `+` 버튼 클릭.
2. **[오디오 출력 캡처 (Audio Output Capture)]** 추가.
3. 장치 선택: `기본 장치` 또는 현재 Zoom 소리가 나오는 `스피커/헤드셋`.

## 4. 오디오 라우팅 (핵심! ⭐)
1. OBS 메인 화면 하단 **[오디오 믹서]**에서 방금 추가한 소스(또는 데스크탑 오디오)의 톱니바퀴 ⚙️ 클릭.
2. **[오디오 고급 설정 (Advanced Audio Properties)]** 클릭.
3. 해당 소스의 **오디오 모니터링** 항목을 `모니터링만 (출력 음소거)` 또는 `모니터링 및 출력`으로 변경.
   - **모니터링만**: 내 스피커에선 안 들리고 AI에게만 보냄 (하울링 방지).
   - **모니터링 및 출력**: 나도 듣고 AI에게도 보냄 (방송용).

## 5. 테스트
1. Zoom이나 유튜브를 틈.
2. `python ai/sound/check_signal.py` 실행.
3. `CABLE Output` 장치 번호를 선택했을 때 게이지(`####...`)가 움직이면 성공! 🎉
